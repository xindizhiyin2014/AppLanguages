# AppLanguages
AppLanguages 是一款支持iOS、macOSX，flutter，android的国际化多语言批量管理工具
- 支持多语言文案导入前的检测
- 支持多语言文案通过excel文件的形式批量导入
- 支持多语言文案通过excel文件的形式批量删除
- 支持多语言文案通过excel文件的形式批量替换
- 支持多语言文案通过excel文件的形式批量导出
